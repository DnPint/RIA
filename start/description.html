Description du projet

Par Daniel Reis, Daniel Pinto et Florian Walter le 12/04/2021 à 20:59

Dans le cadre de notre cours RIA, notre équipe travaille de façon transversale afin de produire un travail mettant en lien l'aspect recherche d'information avec la mise en pratique immédiate, notre projet.

Notre projet consiste à créer un RPG Dungeon Crawler* dans un scénario post-apocalyptique où des personnalités publiques doivent se battre contre le COVID et ses partisans.

Le code source de l'application est disponible sur le github du projet.

* Définition d'après Wikipédia
